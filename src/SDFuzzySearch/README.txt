使用说明

本目录所包含源文件是盛大网络哼唱检索旋律模糊检索模块的源文件，说明如下：

    SDFuzzySearch.h 和 SDFuzzySearch.cpp为盛大网络模糊检索引擎的函数库，提供旋律模糊检索方法；
    
    SFrameMatch.h 和 SFrameMatch.cpp为盛大网络哼唱检索系统的基于帧序列的旋律模糊匹配函数库，供旋律相似度模糊检索库使用；
    
    SNoteMatch.h 和 SNoteMatch.cpp为盛大网络哼唱检索系统的基于音符序列的旋律模糊匹配函数库，供旋律相似度模糊检索库使用；
    
    SSearchCommon.h 和 SSearchCommon.cpp为盛大网络哼唱检索系统的检索通用函数库，供旋律相似度模糊检索库使用；
    
    STypes.h 为盛大网络哼唱检索系统的常量以及结构体定义文件
    
    本目录还附带了一个示例工程（Visual C++ 6）用于编译这些源代码以生成动态连接库程序。当编译生成动态连接库程序之后，用户调用其接口进行旋律的模糊检索，调用方法请参见src\SDHumming的示例程序。
